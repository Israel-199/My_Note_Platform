# MyNotes Platform

MyNotes is a clean and modern note-taking web app that helps users capture, organize, and manage their ideas. It includes OTP-based authentication for secure access and is built using a full MERN + TypeScript stack.

## ✨ Features
- Create, edit, and delete notes
- OTP authentication using Nodemailer
- Organize notes with categories
- Responsive and fast UI
- Secure backend with JWT

## 🛠 Tech Stack
**Frontend:** React, TypeScript, Tailwind CSS  
**Backend:** Node.js (Express), TypeScript, MongoDB, Nodemailer
